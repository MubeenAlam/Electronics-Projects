# Lowpass Filter Design with Inverting Amplifier

**Data:**

f = 30 Hz
C = 470 nF

---

### Lowpass Filter Formula

\[
f = \frac{1}{2 \pi R C}
\]

\[
R = \frac{1}{2 \pi C f}
\]

\[
R = \frac{1}{2 \pi (470n) (30)}
\]

\[
R = 11287.58 \, \Omega \approx 11k \, \Omega
\]

---

### Using the Inverting Amplifier

\[
Gain = A = 9
\]

\[
A = \frac{R_f}{R_{in}}
\]

\[
R_f = 11k \, \Omega
\]

\[
R_{in} = \frac{11k}{9}
\]

\[
R_{in} = 1222 \approx 1200 \, \Omega
\]
